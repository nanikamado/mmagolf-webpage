問題文
=====
$1, 2, \cdots ,N$の順列$P$が与えられます。セパくんは以下の操作を$0$回以上行えます。

- コスト$c$を払い、$1 \leq i \leq N - c + 1$を満たす任意の$i$を$1$つ選び、$P_i, P_{i+1}, \cdots , P_{i+c-1}$を好きな順番に並べ替える。

$P$を昇順に並べ替えるのに必要なコストの最小値を求めてください。

制約
-----
- $2 \leq N \leq 10 ^ 5$
- $1 \leq a_i \leq 10 ^ 9$

入力
-----
入力はすべて整数である。
```
N
a1 a2 ... aN
```

出力
-----
計算結果を一行に出力せよ。


サンプル
=====
```入力1
4
2 1 3 4

```

```出力1
2

```

$a_1, a_2$のみを並べ替えるのが最適です。

```入力2
4
2 1 4 3

```

```出力2
4

```

```入力1
5
1 2 3 4 5

```

```出力1
0

```
