問題文
=====
整数$S, A, B$が与えられます。$x + y = S$を満たす、$A \times x + B \times y$の最大値を求めてください。

制約
-----
- $1 \leq S \leq 10 ^ 9$
- $-10 ^ 5 \leq A, B \leq 10 ^ 5$

入力
-----
入力はすべて整数である。
```
S A B

```

出力
-----
$A \times x + B \times y$の最大値を出力してください。


サンプル
=====
```入力1
10 3 3

```

```出力1
30

```

```入力2
12 -3 2

```

```出力2
24

```
